# CSS Advanced Project

This project is part of the ALX Front-End curriculum.

## Description
This project focuses on styling a webpage using CSS based on a provided HTML structure and a Figma design file.

The goal is to replicate the visual design as accurately as possible using clean and semantic CSS.

## Objectives
- Understand how to link CSS to HTML
- Use CSS to style layouts, text, images, and sections
- Apply Flexbox for alignment and layout
- Follow a design specification from Figma

## Tools Used
- HTML5
- CSS3
- Figma
